# ExplorationMd

A knowledge repository organized by topic. Content is curated from various sources, summarized, and integrated into a navigable knowledge base.

## Topics

- [Agentic Software Development](topics/agentic-software-development/README.md) - AI-powered coding agents, development workflows, and best practices
- [AI Progress](topics/ai-progress/README.md) - Observations on AI capability advancement and cost dynamics
- [AI Tooling](topics/ai-tooling/README.md) - Tools that support AI-assisted work (usage tracking, integrations, utilities)
- [Knowledge Tools](topics/knowledge-tools/README.md) - Note-taking, PKM workflows, and searchable knowledge bases

## Structure

Each topic directory contains:
- `README.md` - Topic overview with links to articles
- `logs.md` - Chronological record of sources
- `*.md` - Individual knowledge articles

## Navigation

Start from any topic above to explore the knowledge base. Every document is reachable through markdown links.
