# ExplorationMd

A knowledge repository organized by topic. Content is curated from various sources, summarized, and integrated into a navigable knowledge base.

## Topics

- [Agentic Software Development](topics/agentic-software-development/README.md) - AI-powered coding agents, development workflows, and best practices
- [AI Engineering Fundamentals](topics/ai-eng-fundamentals/README.md) - Core ML/DL concepts, neural network architectures, and foundational algorithms
- [AI Progress](topics/ai-progress/README.md) - Observations on AI capability advancement and cost dynamics
- [Investment and Finance](topics/investment-finance/README.md) - Investment strategies, market analysis, and financial planning
- [Productivity](topics/productivity/README.md) - Time tracking, measurement systems, and insights from long-term personal data collection
- [Self-Improvement](topics/self-improvement/README.md) - Frameworks, mental models, and strategies for personal growth and life design
- [Stories](topics/stories/README.md) - First-hand accounts from practitioners navigating technological change
- [Technical IT](topics/technical-it/README.md) - Authentication, networking, and systems architecture
- [Tools](topics/tools/README.md) - AI-assisted workflow tools for usage tracking, note-taking, and searchable knowledge systems

## Structure

Each topic directory contains:
- `README.md` - Topic overview with links to articles
- `logs.md` - Chronological record of sources
- `*.md` - Individual knowledge articles

## Navigation

Start from any topic above to explore the knowledge base. Every document is reachable through markdown links.
